descarte las partes que ambos tipos de zapato, con cordones y velcro
siempre van a tener y obtuve los parametros que inclui como listas
para los enum y componentes del producto.

1. Cordones
    1A. Carrillera
    1B. Ojetes
    1C. Color
2. Velcro
3. Almohadilla (Ergonomica y otros tipos)
    3A. Tipo
    3B. Material
4. Tacón
    4A. Altura
    4B. Material

elementos del zapato encajaban mejor como enum.
